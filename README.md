# DataVizLab 简易数据可视化

导入一个`csv`或者`txt`格式的文本文件作为数据源，  
程序将读取其中的数据，并以折线图的形式显示出来。  

## 注意事项
所读取的数据源文件必须为有效的，否则程序无法正常显示。  
第一行的格式必须为：`横坐标轴数据名,纵坐标轴数据名`  
从第二行开始为数据信息：`横坐标节点标题,对应的数值`  

## 格式示例
```txt
月份,月平均气温
1月,13.5
2月,14.8
3月,18.2
4月,22.6
5月,26.0
6月,28.3
7月,29.2
8月,29.1
9月,28.0
10月,25.3
11月,21.0
12月,16.3
```
